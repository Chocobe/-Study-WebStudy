##	jQuery

	>	**$("선택자")** : 요소 선택
	
	>	**$("선택자").html() : 선택한 요소의 body값을 가져온다.
	
	>	**$("선택자").html("값") : 선택한 요소의 body에 값을 넣는다.
	
	>	**$("input태그 선택자").val() : input태그의 입력창에 입력한 값을 가져온다.
	
	>	**$("input태그 선택자").val("값") : input태그의 입력창에 값을 넣는다.
